## ✨ Lumina Note v0.1.0

首个公开发布版本。

### 核心功能

- **多模式编辑器** - 阅读 / 实时预览 / 源码模式自由切换
- **AI Agent 系统** - 支持自动读取、编辑、创建、搜索笔记
- **RAG 语义搜索** - 基于向量数据库的本地知识检索
- **多 LLM 支持** - Anthropic / OpenAI / DeepSeek / Moonshot / Groq / Ollama

### 下载说明

- `lumina-note.exe` 为便携版，下载后直接运行
- 首次启动需选择或创建笔记库文件夹
- 使用前请在设置中配置 AI 提供商的 API Key

### 系统要求

- Windows 10/11 (x64)
